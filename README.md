## 개요

FlashEPUB은 별도 설치 없이 브라우저에서 바로 실행되는 오프라인 EPUB 생성기입니다.  
학습 데이터를 담은 표 형식 파일을 불러오면, Xteink X4에 최적화된 플래시카드 EPUB을 즉시 만들어 줍니다.'

**클로드 AI를 이용하여 제작하였습니다.**

---

## 주요 기능

- **다양한 입력 형식 지원** — CSV, XLSX/XLS, TXT 파일 파싱
- **X4 미리보기** — 변환 전에 X4 화면 레이아웃 그대로 카드 내용을 미리 확인
- **표지 자동 생성** — 책 제목과 카드 수가 표시된 표지 이미지(PNG) 자동 생성
- **힌트 컬럼 지원** — 질문 페이지에 선택적 힌트 표시
- **컬럼 자유 지정** — 헤더명 또는 열 번호로 질문·정답·힌트 컬럼 직접 지정
- **구분선 스타일 선택** — `....` / `----` / `====` 중 선택
- **완전 오프라인 동작** — 서버 전송 없이 브라우저 내에서만 처리

## 사용 방법

### 1. 파일 열기

`index.html` 파일을 크롬 등 최신 브라우저로 열면 바로 실행됩니다.

### 2. 학습 파일 업로드

드래그 앤 드롭 또는 클릭으로 파일을 선택합니다.

| 형식 | 설명 |
|------|------|
| **CSV** | UTF-8 인코딩, 첫 행 헤더 권장 |
| **XLSX / XLS** | 첫 번째 시트의 첫 행을 헤더로 사용 |
| **TXT** | 탭·파이프(`\|`)·쉼표 구분, 헤더 행 선택 가능 |
| **APKG** | Anki 덱 파일 (최대 3,000장) |

### 3. 컬럼 설정

| 항목 | 기본값 | 설명 |
|------|--------|------|
| 질문 컬럼명 / 열 번호 | `question` | 헤더 이름 또는 열 번호(1부터 시작) |
| 정답 컬럼명 / 열 번호 | `answer` | 헤더 이름 또는 열 번호 |
| 힌트 컬럼 (선택) | `hint` | 없으면 비워두세요 |
| 책 제목 | `플래시카드 학습` | EPUB 메타데이터 및 표지에 사용 |
| 저자명 | `FlashEPUB` | EPUB 메타데이터에 사용 |
| 구분선 | `....` | 질문과 정답 사이 구분 문자 |

### 4. 미리보기 확인

파일이 로드되면 오른쪽 **X4 미리보기** 패널에서 카드를 탐색할 수 있습니다.

- `← / 다음 →` 버튼으로 카드 이동
- **뒤집기** 버튼으로 질문 ↔ 정답 전환
- 왼쪽 목록에서 특정 카드를 클릭해 바로 이동

### 5. EPUB 생성 및 다운로드

**"X4용 EPUB 생성"** 버튼을 클릭하면 변환이 시작되고, 완료 후 **EPUB 다운로드** 버튼이 나타납니다.

## EPUB 구조

생성된 EPUB은 카드 한 장당 두 페이지로 구성됩니다.

```
표지 (cover.xhtml)
  └─ 카드 1 — 질문 페이지 (q001.xhtml)
  └─ 카드 1 — 정답 페이지 (a001.xhtml)
  └─ 카드 2 — 질문 페이지 (q002.xhtml)
  └─ 카드 2 — 정답 페이지 (a002.xhtml)
  └─ ...
```

각 페이지는 모노스페이스 폰트와 텍스트 레이아웃으로 구성되어 X4의 전자잉크 화면에 최적화되어 있습니다.

## 입력 파일 예시

### CSV

```csv
question,answer,hint
대한민국의 수도는?,서울,반도 국가
물의 화학식은?,H2O,수소와 산소
```

### TXT (탭 구분)

```
question	answer	hint
광합성이란?,빛 에너지를 화학 에너지로 변환하는 과정	식물
```

## 동작 환경
